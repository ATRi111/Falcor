- Hybrid LOD 的每帧 route resolve 不能逐实例调用 `set_geometry_instance_route()`；那样会重复清 filtered draw cache 并频繁上传实例缓冲。应改成 batched `set_geometry_instance_routes()`，一帧只做一次 scene route 提交。
