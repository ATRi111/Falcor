# Future Hybrid Renderer Vision

## 定位

- 本文件只描述最终渲染方案的愿景。
- 本文件不给实施步骤，不拆阶段，不写具体措施。
- 需要实现路径时，看 `E:\GraduateDesign\Falcor_Cp\.FORAGENT\plan4.md`。

## 最终目标

- 近景以 mesh 渲染为主。
- 远景以 voxel 渲染为主。
- 中景由 mesh 与 voxel 共同参与，并以平滑过渡完成连接。
- 用户在正常观察时，不应一眼看出画面在某个距离突然切换了渲染路径。

## 我期望的最终画面

- 近景拥有 mesh 的几何细节、边缘质量和材质表达。
- 远景保持当前 voxel 主线的稳定感、整体轮廓和统一风格。
- 中景在相机前后移动时不会突然变亮、变黑、变灰，也不会突然变成另一套阴影硬度。
- 如果不打开 debug 视图，观察者最好感觉不到 mesh / voxel 的边界。

## 我期望的最终系统形态

- mesh 和 voxel 不是“谁完整替换谁”，而是长期共存。
- voxel 继续承担远景、粗 LOD 和稳定整体观感。
- mesh 负责近景、局部细节和更高质量的表面表达。
- 中景不是硬切，而是基于可比较的 shading 信号做融合。

## 我期望的最终一致性

- mesh direct lighting 与 voxel direct lighting 在能量、阴影倾向和整体明暗关系上保持同一口径。
- mesh AO 与 voxel AO 在接触暗化、强度和空间尺度上保持同一口径。
- background、tone mapping、整体对比度和风格收尾必须统一。
- 最终合成后的画面应该让人感觉是“一套渲染器”，而不是“两套结果拼起来”。

## 我期望的最终混合原则

- 中景过渡必须平滑、稳定、可控。
- 混合区不应因轻微相机移动而抖动或翻转主次。
- 混合的目标不是平均两张图，而是保持视觉连续性。
- 当 mesh 与 voxel 不一致时，系统应优先保证画面自然，而不是机械地按距离切换。

## 我期望保留的主次关系

- 当前 Stage4 voxel 主线仍是整体视觉锚点。
- mesh 路径的价值是补足近景，不是推翻当前 voxel 风格。
- 最终 hybrid 方案的成功标准，不是 mesh 占比更高，而是近景更精细、远景更稳定、中景不穿帮。
